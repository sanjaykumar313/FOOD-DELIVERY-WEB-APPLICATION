\# Food Delivery Web Application



\## Overview

A full-stack Food Delivery Web Application developed using Java Servlets, JSP, Maven, JDBC, MySQL, HTML, CSS, and JavaScript.



\## Features

\- User Registration and Login

\- Restaurant Listing

\- Menu Management

\- Cart Management

\- Order Placement

\- Order Tracking

\- Restaurant Management



\## Technologies Used

\- Java

\- JSP \& Servlets

\- JDBC

\- MySQL

\- Maven

\- HTML

\- CSS

\- JavaScript



\## Project Structure

src/main/java

src/main/webapp



\## Setup

1\. Clone the repository

2\. Import as Maven Project

3\. Configure MySQL Database

4\. Run on Apache Tomcat
